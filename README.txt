BUILDING:
	To build the project, run make.
	The executable will be compiled to the bin/ directory.
	The project writeup will be in the docs/ directory.

RUNNING:
	To run the project, run the executable in the bin/ directory.
	Without arguments, the program will repeatedly play standard tic-tac-toe until the user kills it with Ctrl+C.
	To play 9-board tic-tac-toe, pass an argument to the executable.
	If the first argument to the executable is a lowercase or uppercase s, the program will print statistics about the game.
